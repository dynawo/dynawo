Detailed Description

Trace static class.

Trace static class is used to generate log files and configure them. It is also the only interface for logging streams in Dynawo.
Traces generated before any call to init() static function are ignored.

Add a log to logging core.

Add a message of given severity level with a given tag to the logging core. If tag=="" , no tag is added the log. The logging core get all the messages and each sink get info from the core that a log has arrived and is responsible for filtering.

Parameters
slv: Severity level of the log.
tag: Tag added to the log, can be used as a filter in logging sinks.
message: Message to log.
