Detailed Description

Voltage measurement utilities model class.

This model connects to a dynamic number of buses and outputs some interesting values computed from the input voltage levels. Outputs currently included: -> Minimum voltage level -> Maximum voltage level -> Average voltage level -> Where minimum voltage level is coming from -> Where maximum voltage level is coming from
